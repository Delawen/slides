Un buen recurso para aprender todo lo necesario sobre cartografía es el Libro Libre de SIG Libre[1].

Todo el software que necesitáis para montar un stack geográfico podéis encontrarlo bajo el paraguas de OsGeo[2]. Para probarlo fácilmente está el DVD Live de OsGeo[3] con prácticamente todo el software instalado.

Proyectos consolidados:

 * deegree
 * geomajas
 * GeoMoose
 * GeoServer
 * Mapbender
 * MapBuilder
 * MapFish
 * MapGuide Open Source
 * MapServer
 * OpenLayers

Aplicaciones de Escritorio (operaciones pesadas con UI)
 * GRASS GIS
 * Marble
 * QGIS

Librerías GeoEspaciales (operaciones aún más pesadas sin UI)
 * FDO
 * GDAL/OGR
 * GEOS
 * GeoTools
 * OSSIM
 * PostGIS

Catálogo de Metadatos (portal de datos)
 * GeoNetwork
 * pycsw
