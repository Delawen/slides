Instalación

Por simplificar, usaremos Linux

(de todas formas otro SO no sería serio)

Necesitamos:

  • Un ordenador con Linux (ejem. Debian)
  • Internet
  • Java (mvn)
  • Postgresql (opcional)
  • 27 minutos

Instalación

Para valientes:

$ git clone --recursive git://github.com/geonetwork/core-geonetwork.git master

$ cd master

$ git checkout master

$ git submodule update --init

$ mvn clean install

Para conservadores:

Sourceforge
